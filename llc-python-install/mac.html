Python 3 setup

  1. Download the learner files: http://bitly.com/llc-python (zip)
  2. Extract the zip
  3. Install latest Python 3: https://www.python.org/downloads
  4. Install Pycharm Educational Edition: https://www.jetbrains.com/pycharm-educational
  5. Run Pycharm and open the project folder you unzipped.
  6. Configure Pycharm to use Python 3 instead of 2.7.
    • Select Pycharm > Preferences
    • Select the project and then Project Interpreter
    • Depending on your setup you may or may not have a Project Interpreter selected from the drop down menu.
    • Beside the drop down is a little cog which you can select and click add
    • Another dialog should pop up now called Add Python Interpreter
    • Select System Interpreter on the left hand side and then select the Python 3.6 you installed in the previous setup. This will typically be: /usr/local/bin/python3.6.
    • Click OK through all the windows and you should see something as follows for the Preference dialog
    • Once you are back into the main project window, click Python console then the rerun button
    • You should see the version in the console like so (3.6.x or similar):
  7. You're good to go!
